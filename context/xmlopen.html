XML OPEN VBA Code

파일 선택 창을 통해 XML 파일을 불러오고 데이터를 파싱하여 직접 출력하는 매크로 소스코드입니다.

Sub xmlOpen() Dim fd As Office.FileDialog Set fd = Application.FileDialog(msoFileDialogFilePicker) With fd .Filters.Clear .Title = "Select Xml File" .Filters.Add "XML File", "*.xml", 1 .AllowMultiSelect = False If .Show = True Then xmlFileName = .SelectedItems(1) ' MsgBox xmlFileName ' Process Xml files Dim xDoc As Object Set xDoc = CreateObject("MSXML2.DOMDocument") xDoc.async = False: xDoc.validateOnParse = False ' Get Root Node Set Products = xDoc.DocumentElement For Each Product In Products.ChildNodes Debug.Print "key : " & Product.ChildNodes(0).Text Debug.Print "Text: " & Product.ChildNodes(1).Text Debug.Print "---" Next Product End If End With End Sub
