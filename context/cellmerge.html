CellMerge VBA Code

선택한 영역에서 동일한 값을 가진 셀들을 자동으로 병합해주는 매크로 소스코드입니다.

Sub testCellMerge() Dim rStart As Range Dim rend As Range Dim index As Integer Application.DisplayAlerts = False Set rStart = Selection.Cells(1) For index = 1 To Selection.Cells.Count If Selection.Cells(index) <> Selection.Cells(index + 1) Then Set rend = Selection.Cells(index) Range(rStart, rend).Merge Set rStart = Selection.Cells(index + 1) End If Next Application.DisplayAlerts = True End Sub
